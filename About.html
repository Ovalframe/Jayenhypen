LIKES AND DISLIKES

  • เจย์ชอบเสื้อผ้าและแฟชั่นโดยรวม เขามีรายการของตัวเองในช่อง YouTube ของ Enhypen ซึ่งเขาแบ่งปันเคล็ดลับการแต่งตัวและแฟชั่นโดยได้รับความช่วยเหลือจากสมาชิก
  • เขาชอบทำอาหาร กินอาหารอร่อยๆ และทานอาหารตามร้านอาหาร บางครั้งเขาก็พาสมาชิกไปทานอาหารที่ร้านที่เขาชอบ ( ที่มา ) ( ที่มา 2 )
  • เขาชอบกลิ่นหอมและน้ำหอมและมีคอลเลกชันมากมาย ( ที่มา )
  • เจย์ชอบแกง อาหารทะเล และหอยเป็นพิเศษ ( ที่มา )
  • เขาไม่ชอบกาแฟและชอบชาดำหรือชานมมากกว่า ( ที่มา )
  • ในชีวิตส่วนตัวของเขา เขาชอบที่จะทำอะไรสบายๆ และใช้เวลาอย่างไม่เร่งรีบ ( ที่มา )  
  • เจย์ชอบนอนหลับและหลับง่าย เขามักจะหลับในตอนต่างๆ ของ EN-O'CLOCK
